# KanaKana

## Overview
An iPhone application that converts Japanese Kanji letters input -> Japanese Kana letters output.

## Requirement
- macOS
- Xcode 12.3

## Usage

## Features
- converts Japanese Kanji to Kana
- e.g.
  - input: `漢字`
  - output: `かんじ` or `カンジ`
